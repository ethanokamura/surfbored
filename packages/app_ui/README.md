# App UI libary
handles custom UI components and themes for the app

```
app_ui/lib/src/
  ├── buttons.dart          # Custom buttons for the app
  ├── colors.dart           # Custom colors for the app
  ├── constants.dart        # Constant values including app strings
  ├── extensions.dart       # Extensions for the UI
  ├── images.dart           # Custom image containers
  ├── pop_ups.dart          # Modals
  ├── theme.dart            # Defines the light and dark mode as well as handles changes
  └── widgets.dart          # Misc reusable widgets
```
